## Install Rust
- install by rustup
`curl --proto '=https' --tlsv1.3 https://sh.rustup.rs -sSf | sh`
- update
`rustup update`
- uninstall
`rustup self uninstall`

## Build & Run
- new project
`cargo new someproject`
- check before building
`cargo check`
- build
`cargo build`
- build and run
`cargo run`
- build release
`cargo build --release`
